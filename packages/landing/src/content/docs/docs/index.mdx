Zedra provides remote control for AI coding agents. Your agents run on your own machine, controlled from your phone.

Developers want to run coding agents from their phone, and we need more than just a simple chat UI. Zedra provides a developer workspace designed for mobile, where you can browse files, read code and documents, review changes, run agents and customize your own workflow.

## Features

- Run any coding agent over remote terminal.
- File browser, code editor, markdown preview, git view.
- Agent usage, history, notifications.
- Customizable agent workflow and features.
- GPU-accelerated rendering powered by Zed's GPUI.
- Direct connection between devices over QUIC/UDP powered by Iroh.

## How it works

Zedra runs a desktop daemon on your computer. Your phone connects to it over a secure tunnel with TLS 1.3, over QUIC/UDP powered by Iroh. QR scanning handles the first-time pairing to register your devices. After that, the connection is kept across reconnects by PKI auth. Remote control features are defined in [Zedra Protocol](https://github.com/tanlethanh/zedra/blob/main/docs/PROTOCOL_SPECS.md), which the host and client app use to communicate.
