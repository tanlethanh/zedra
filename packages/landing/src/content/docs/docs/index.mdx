Zedra is a real dev workspace on your phone. Your agents run on your own machine; the phone gets the tools to do actual work with them.

Steering an agent is half the job — the other half is reading what it did, and a chat UI is not enough for that. Zedra is a developer workspace designed for mobile: browse files, read code and documents, review changes, run any agent in a real terminal, and shape your own workflow.

## Features

- Run any coding agent over remote terminal.
- File browser, code editor, markdown preview, git view.
- Agent usage, history, notifications.
- Customizable agent workflow and features.
- GPU-accelerated rendering powered by Zed's GPUI.
- Direct connection between devices over QUIC/UDP powered by Iroh.

## How it works

Zedra runs a desktop daemon on your computer. Your phone connects to it over a secure tunnel with TLS 1.3, over QUIC/UDP powered by Iroh. QR scanning handles the first-time pairing to register your devices. After that, the connection is kept across reconnects by PKI auth. Remote control features are defined in [Zedra Protocol](https://github.com/tanlethanh/zedra/blob/main/docs/PROTOCOL_SPECS.md), which the host and client app use to communicate. For what is encrypted and what our servers can see, read [Security & architecture](/docs/security/).
