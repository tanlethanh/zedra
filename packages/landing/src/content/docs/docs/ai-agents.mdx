Zedra runs terminal AI coding agents on your desktop and lets you drive them from
your phone — start a run, watch progress, review diffs, and get notified when an
agent needs you.

## Set up your agent

Run the matching `zedra setup` command once. It installs the skills and hooks
the agent needs to talk to Zedra, then start a session from inside the agent.

<Tabs>
  <TabItem label="Claude Code">
```sh
# Configure Zedra skills and hooks for Claude Code
zedra setup claude
```
Then, inside Claude Code, reload plugins and start Zedra:
```text
/zedra-start
```
  </TabItem>
  <TabItem label="Codex">
```sh
# Configure Zedra skills and hooks for Codex
zedra setup codex
```
Then, inside Codex, reload skills and start Zedra:
```text
$zedra-start
```
  </TabItem>
  <TabItem label="OpenCode">
```sh
# Configure Zedra skills and hooks for OpenCode
zedra setup opencode
```
Then, inside OpenCode, reload skills and start Zedra:
```text
/zedra-start
```
  </TabItem>
</Tabs>

## What you get on mobile

- **Agent detection.** Zedra recognizes the running agent and shows its icon on
  the terminal card, so you can tell at a glance what's running where.
- **Notifications.** When an agent finishes a turn or needs your input, Zedra can
  notify you on your phone.
- **Add to Chat.** Select code in the editor or a file preview and send it to the
  running agent as context.

## Other agents

Zedra detects a wide range of terminal coding agents beyond Claude Code, Codex,
and OpenCode. Any supported agent running in a Zedra terminal is detected
automatically and shows its icon — no extra setup beyond starting it in a
workspace the app is connected to.
